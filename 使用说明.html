🐱 日报喵 (DailyBot) 使用说明

自动从 Git 拉取你的代码提交 → AI 总结 → 推送到飞书/企业微信

🤔 这是什么？

日报喵是一个帮你自动写日报的工具。每天下班时，它会：

1️⃣ 从 GitLab/GitHub/Gitee 拉取你今天的所有代码提交

2️⃣ 交给 AI 自动总结成 readable 的日报内容

3️⃣ 推送到飞书或企业微信——你打开手机就能看到

你再也不用下班前翻 commit 记录、复制粘贴写日报了。

📋 准备工作

你需要：

✅ 一台 Windows 电脑（本项目专为 Windows 设计）

✅ 一个 AI 的 API Key（或者安装免费的 Ollama）

✅ 一个 Git 仓库（你的代码在 GitLab/GitHub/Gitee 上）

✅ 一个 飞书或企业微信（用于接收日报）

🚀 第一步：配置

1

进入项目文件夹

打开"此电脑"，进入 D:\daily-bot

2

配置环境变量

找到 .env 文件，右键 → 用记事本打开

把你用的 AI 的 API Key 填进去，比如：

# 去掉前面的 # 号，把 xxx 替换成你的 Key
DEEPSEEK_API_KEY=sk-你的DeepSeekKey

如果想用免费的本地 AI，看后面的"免费方案"。

3

配置代码仓库

找到 config/config.yaml 文件，用记事本打开

找到 crawler_sources 这一段，配置你的 Git 仓库：

# GitHub 示例（把你自己的仓库地址填进去）
github:
  enabled: true
  token: "ghp_你的GitHubToken"
  repos:
    - path: "你的用户名/你的仓库名"
      branch: "main"
4

配置推送平台（选一个就行）

在 config.yaml 中找到 enabled_workflows：

# 二选一：用飞书还是企业微信
enabled_workflows: ["feishu"]      # 用飞书
# enabled_workflows: ["wecom"]    # 用企业微信

然后在 platforms 段填上对应的 App ID 和 Secret。

💰 免费方案（推荐新手）

不想花钱买 API Key？用 Ollama 在你自己电脑上跑 AI，完全免费：

1

去 ollama.ai 下载并安装 Ollama

2

打开命令提示符（Win+R → 输入 cmd → 回车），运行：

ollama pull qwen2.5:7b

等待下载完成（约 4GB，需要一些时间）

3

在 config.yaml 中把 AI 模型设为 ollama：

# 找到你启用的平台，把 ai_model 改成 ollama
platforms:
  feishu:
    ai_model: "ollama"
💡 提示：Ollama 在你自己的电脑上运行，数据不会传到第三方，隐私安全。缺点是生成速度比云端 AI 慢一些。

▶️ 第二步：运行

1

双击运行

找到 run.py 文件…… 不过 .py 文件不能直接双击运行。

你需要先打开命令提示符：

按键盘上的 Win + R → 输入 cmd → 回车

在弹出的黑窗口里输入：

cd D:\daily-bot
python run.py

按回车，它就开始运行了！

2

查看结果

运行结束后，打开你的飞书或企业微信，应该就能看到今天的日报了。

黑窗口里也会打印出 AI 生成的日报内容。

💡 提示：第一次运行可能会安装浏览器驱动，稍等一会儿就好。
⚠️ 如果报错："python 不是内部或外部命令"——说明你没装 Python。去 python.org 下载安装，安装时记得勾选 "Add Python to PATH"。

⏰ 第三步：每天自动运行

不想每天手动跑？让它每天下班时间自动运行：

在黑窗口里输入：

cd D:\daily-bot
python run.py --scheduler

它会注册一个 Windows 定时任务，默认每天 18:20 自动运行。

以后你每天下班打开手机就能看到日报了。

想改时间？打开 config/config.yaml，找到：

scheduler:
  default_time: "18:20"    # 改成你希望的时间

❓ 常见问题

Q: 运行报错 "No module named xxx"

缺少依赖包。在命令提示符中运行：

cd D:\daily-bot
pip install -r requirements.txt

Q: 如何查看当天的日报内容？

运行后黑窗口里会打印出 AI 总结的日报。也可以看日志文件：

cd D:\daily-bot
notepad logs\dailybot_2026-06-24.log

Q: 不想用了怎么卸载？

cd D:\daily-bot
python run.py --uninstall

这会删除定时任务和开机自启动。

Q: 如何查看当前配置了哪些定时任务？

cd D:\daily-bot
python run.py --status
